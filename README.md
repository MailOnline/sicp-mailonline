# sicp-mailonline

Yet another SICP follow along.

## Usage

```
lein generate exercise 1-1 sicp-mailonline midje
lein generate paragraph 1-1 sicp-mailonline midje
```

will generate a ready to play test/src combo for an exercise or paragraph.
